# FPY

## Code Snippets

### Basic Variables
```haskell
let x = 1
x
```
```txt
1
```

### Arithmethic Expressions
```haskell
let x = 1
let y = 2
x + y / 4
```
```txt
1.5
```

### Strings
```haskell
let myString = "Hello, "
myString + "World!"
```
```txt
Hello, World!
```

### Lambdas
```haskell
let add = &x &y x + y
add 1 2
```
```txt
3
```
